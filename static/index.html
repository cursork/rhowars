      Rhowar Lineup

        Rhobot API Reference 
        ── Init (niladic) ──────────────
Returns: state (any namespace)

── Tick input ──────────────────
input fields:
  loc        2-vec: x y position
  direction  current heading (degrees)
  turret     current turret angle (degrees)
  hp         current hit points
  arena      2-vec: width height
  visible    K×3 matrix (or empty):
               [;0] distance
               [;1] angle offset from turret
               [;2] type: 0=rhobot 1=bullet
  state      your state from last tick
  cfg        namespace with game config:
               .botSpeed    movement per tick
               .botHP       starting hit points
               .botRadius   collision radius
               .bulletSpeed bullet speed per tick
               .bulletDamage damage per hit
               .visionRange vision distance
               .visionHalfAngle half-width of vision cone
               .cooldown    ticks between shots
               .maxTurns    match turn limit

output fields (all optional, ⍬ = no change):
  direction  new heading (degrees)
  turret     new turret angle (degrees)
  fire       1 to fire, 0 to not
  speed      0 to stop, 1 to move
  state      updated state
        Turn: 0
